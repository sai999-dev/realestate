Field Descriptions

Field Type Description Required
id integer Unique identifier for the record Auto-generated
name string Customer's full name Yes
email string Valid email address Yes
contact string Phone number Yes
needs string Detailed real estate requirements Yes
property_type string Type of property (Residential, Commercial, Industrial, Land) No
budget_range string Price range preference No
preferred_location string Desired location or area No
timeline string When property is needed No
additional_details string Extra requirements or notes No
industry string Customer industry (Home Health and Hospice, Finance, Insurance, Handyman Services) No
zipcode string Customer zip code No
submitted_at string ISO 8601 timestamp of submission Auto-generated

cURL Example

curl -X GET "https://your-domain.com/api/schema" \
  -H "Accept: application/json"

Integration Notes

  • All API endpoints return JSON formatted data
  • Timestamps are in ISO 8601 format (UTC)
  • Customer data is stored in Supabase database
  • Real-time data synchronization with Supabase backend
  • In production, implement proper authentication and authorization
